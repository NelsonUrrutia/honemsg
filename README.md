# honemsg

### Sharpen your messages with a local LLM

> `honemsg` is a CLI tool that improves your draft messages and give you suggestions using a local LLM. Set the context and tone, get back a sharpen text.

## Features

- **Context**
  - Slack message
  - E-mail
  - Documentation
  - General text

- **Tone**
  - Professional
  - Technical
  - Instructions

- **Grammar**
  - Concise
  - Clarity
